# Frontend Notes

The React frontend is located in `client/` and is built with Vite.

Implemented pages/sections:

- Shop/product catalogue
- Search and category filter controls
- Login/register forms
- Shopping cart
- Mock checkout form
- Purchase history
- Admin dashboard with product management and purchase records

Accessibility/responsiveness:

- Semantic headings and labelled form fields
- Keyboard-focusable buttons and inputs
- ARIA labels for main product actions
- Mobile-friendly CSS grid layout
